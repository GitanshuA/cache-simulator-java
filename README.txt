This repo contains files for the Java Assignemnt that simulates the Cache Memory.

To build the program, execute the command 'make' in the terminal in the directory where the files are kept.

Once the program has been built it can be run in the terminal using the following command and arguments:-
Usage: java Main CacheSize(P) Associativity(W) BlockSize TraceFilePath

NOTE:- The program takes the block requests input from the Trace file in HEXADECIMAL format.
